Terminal

Runs commands as root on the adapter via term.cgi. Full shell syntax (pipes, redirects). Working directory persists between commands. Avoid interactive (top, vi) or long-running (ping with no count) commands — there is no live streaming, and a request that never returns is aborted after 60s.

⚠

Danger — Root Shell

This terminal executes commands as root directly on the adapter, with no sandbox and no undo. A single mistyped command can corrupt the firmware, erase configuration, brick the adapter, or drop your connection.

  • Commands run immediately — there is no per-command confirmation.
  • Destructive operations (rm, dd, mkfs, reboot, killing services) take effect at once.
  • Only run commands you understand. When in doubt, don't.
